# ut-plugin-payment-qrpay — rules

Payment-type plugin for Universal Till. Docs repo ADRs are binding:
ADR-0001 (runtime none/wasm — never add an unnecessary binary), ADR-0002
(entry types), ADR-0009 (this repo's naming + pipeline). Manifest contract:
docs repo reference/plugin-manifest.md. Bump the manifest version and tag
`v<version>` to release; never publish by hand.
